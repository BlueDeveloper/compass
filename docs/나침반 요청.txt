요청사항

해당 프로젝트는 모바일 웹 페이지에서  특정좌표를 대상으로 나침반으로 가르켜주는 기능을 서비스한다
현재 Next.js 프로젝트 껍데기는 이미 생성되어 있으며, MVP 기준으로 아래 유틸들을 구현한다.

1. MVP 기능 목록
1-1. 나침반으로
현재 작성되어있는 좌표를 대상으로 모바일에서 웹사이트 접속 시 나침반으로 방향을 가르킨다.

2. 기술 및 구조 제약사항
Next.js App Router 기준
백엔드는 사용하지 않음
(Spring Boot, DB, 인증 모두 제외)
가능한 한 Client-side 처리 우선
필요 시 Next.js API Route 사용 가능
외부 라이브러리 사용 가능하나 과도한 의존성 지양

3. MVP 비기능 요구사항 (중요)
로그인 / 회원가입 ❌
사용자 데이터 저장 ❌
히스토리 / 로그 저장 ❌

4. UX / 화면 구성 가이드
화면중앙에 나침반
사용자에게 위치정보 활용동의 받기

6. 에이전트 파일 생성 요구사항
프로젝트 루트에 .agent.md 파일 생성
다음 내용을 포함할 것:
프로젝트 목적
MVP 기능 목록
기술 스택 및 제약사항
향후 확장 시 고려사항
이후 작업 시 해당 파일을 항상 참조 기준으로 사용
=============================================================================================
요청사항

1. 아이폰/안드로이드 둘 다 모바일 기반 위치정보 확인 완료
2. 모바일 디바이스의 방향 센서를 활용하여 실시간 방향 안내가 되지 않음(안드로이드는 나침반이 불안정하게 계속 움직임, 아이폰(사파리)는 나침반이 움직이지 않음
